使用说明
  • 0 0 3 * * ? 每天3点执行
  • 0 5 3 * * ? 每天3点5分执行
  • 0 5 3 ? * * 每天3点5分执行，与上面作用相同
  • 0 5/10 3 * * ? 每天3点的 5分，15分，25分，35分，45分，55分这几个时间点执行
  • 0 10 3 ? * 1 每周星期天，3点10分 执行，注：1表示星期天
  • 0 10 3 ? * 1#3 每个月的第三个星期，星期天 执行，#号只能出现在星期的位置
参数说明
  • * 第一位，表示秒，取值0-59
  • * 第二位，表示分，取值0-59
  • * 第三位，表示小时，取值0-23
  • * 第四位，日期天/日，取值1-31
  • * 第五位，日期月份，取值1-12
  • * 第六位，星期，取值1-7，星期一，星期二...，注：不是第1周，第二周的意思
    另外：1表示星期天，2表示星期一。
  • * 第7为，年份，可以留空，取值1970-2099
